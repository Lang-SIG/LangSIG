## LangSIG

[![Read the Docs](https://readthedocs.org/projects/lang-sig/badge/?version=latest)](https://lang-sig.readthedocs.io/en/latest/)

#### 사용법 1
- $ cd docs
- $ pip install -r requirements.txt
- $ make html

#### 사용법 2 ([rye](https://rye.astral.sh/) 사용)
- $ rye sync
- $ cd docs
- $ make ryehtml
